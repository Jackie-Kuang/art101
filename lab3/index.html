Challenge

Problems

When I was writing the code to create the list, I forgot how to properly write it. In each line I wrote a ul tag in front of each element so in the website, it created a horizontal list of links, not a proper vertical list. So I looked back to my notes and realized within a ul tag, I have to write a li tag to tell that each item is a list item which solved my problem.

For the other things like referencing the images and creating the list to link our pages it was pretty easy to add so overall I felt really comfortable.
